Дизайн сайта

Подготовим дизайн сайта к разработке. Сделаем дизайн макеты в 3-5 размерах, UI-kit, палитры и типографику.

Дизайн мобильных приложений

Спроектируем навигацию и архитектуру мобильного приложения. Сделаем дизайн приложения и UI-kit для разработки.

Дизайн система

Наведём порядок в дизайне, определим общие принципы, правила колористики и типографики. Систематизируем и автоматизируем, что возможно.

Кликабельный прототип

Сделаем точный кликабельный прототип проекта, чтобы протестировать интерфейс, найти ошибки и улучшить до разработки.

Редизайн сайта

Обновим дизайн сайта или приложения, систематизируем дизайн проекта, сделаем редизайн с учётом новых функций.

Аудит Usability

Соберём и изучим конкурентов, проведём аудит по трендам UI UX, составим список ошибок и рекомендаций по улучшению интерфейса.

UI UX дизайн

Упростим жизнь пользователя, протестируем интерфейс до разработки, сделаем удобнее по мнению самих пользователей.

UX тесты на пользователях

Проведём интервью, получим инсайты от самих пользователей. Изучим, что нужно людям, и решим проблемы в интерфейсе.

ТЗ и документация

Составим полное ТЗ на разработку, которое содержит User Story, ACL, функциональные требования, тест-кейсы и требования к UI.

Разработка MVP

Поэтапно реализуем API и бизнес-логику для MVP вашего проекта. Стек: React.js + Node.js (PostgreSQL).

Дизайн на аутсорсе

Работаем как дизайн-отдел на аутсорсе. Делаем задачи по дизайну, проводим исследования и систематизируем процесс.
